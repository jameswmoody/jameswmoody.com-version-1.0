Stripe Integration

So...I figured since I've already put together a full working example of Braintree's Drop-In UI, I might as well try to do the same thing with Stripe's Checkout. Again I decided to go with a PHP integration which actually proved to be a bit more difficult (for me at least) than expected. Stripe's PHP docs assumes you are are using Composer to install their library, which is fine, but is a bit less clear about a manual install. I'm sure this all really just comes down to my lack of more advanced experience with PHP but proved to be challenging none the less. After a bit of tinkering I was finally able to get both my client and server files to work as expected to get an extremely barebones checkout flow working through Stripe's test mode. Over the next week or so I plan to expand on this and see if I can build out something similar to my Braintree integration found here.
