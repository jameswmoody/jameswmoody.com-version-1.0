Portfolio

I'll be posting recent stuff I've worked on here. Mostly these are web development projects I've built but every once in a while you might see something else here. Keep checking back!

TabOut

A marketing website for a geolocation-based bar tab manager mobile app. Fully responsive design via Bootstrap.

Magic 8 Ball

My take on the classic magic 8 ball. It uses a bit of Javascript to make it work, along with some fancey CSS to style it.

Shopping List

Basic shopping list that can add new items, check off items, reorganize items, and double-click to remove these items. It uses a bit of jQuery to make it all work.

Braintree Integration

An e-commerce integration using Braintree Payment's Drop-In UI. Braintree's PHP SDK is used for the backend.

WaxTracks

Commercial website for a record store. Features a modern, responsive design via Bootstrap. I'd like to make the search feature fully functional at some point in the future.

Personal Blog App

Here's a web application built with Ruby on Rails that mimics a personal blog. I've left the create, edit, and delete fuction unrestricted so anyone can play around with it.
